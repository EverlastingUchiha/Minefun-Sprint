# Minefun-Sprint

Sprint with W without pressing Shift on Minefun.io! No need to hold two keys anymore.

## Features
- **Auto-Sprint:** Just hold `W` to run instantly.
- **Smart Detection:** Automatically turns off when you type in chat (`INPUT` or `TEXTAREA`).
- **Smooth Gameplay:** Dispatches events directly to the game canvas for zero lag.

## Installation
1. Install the [Tampermonkey](https://tampermonkey.net) extension in your browser.
2. Click on the **sprint.user.js** file inside this repository.
3. Click on the **Raw** button to install it directly!
